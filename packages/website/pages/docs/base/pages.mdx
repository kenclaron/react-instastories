# Pages

## Description

The `Pages` component manages and renders a series of pages within a [story](/docs/base/story).
It leverages multiple hooks to access and manage context values related to the viewer, [pages](/docs/base/contexts/pages), [timer](/docs/base/contexts/timer), and [stories](/docs/base/contexts/stories).
Additionally, it uses a hook to order and render its children elements, ensuring the correct sequence and transitions between pages.

## Properties

### Component

| Prop        | Type                                                               | Description                                                 |
| ----------- | ------------------------------------------------------------------ | ----------------------------------------------------------- |
| `as`        | `React.ElementType`                                                | Specifies the component type to render (by default, `div`). |
| `...events` | [`PaginationEvents`](/docs/base/hooks/pagination#component-events) | Optional callback events for component.                     |
| `...props`  | `React.HTMLAttributes`                                             | Standard HTML attributes for an HTML element.               |

## Contexts Available

- [`useViewerContext`](/docs/base/contexts/viewer)
- [`useTimerContext`](/docs/base/contexts/timer)
- [`useStoriesContext`](/docs/base/contexts/stories)
- [`usePagesContext`](/docs/base/contexts/pages)
- [`useConfig`](/docs/base/hooks/config)

## Usage

```tsx copy
<Pages>
  <Page>{...content}</Page>
  <Page>{...content}</Page>
  <Page>{...content}</Page>
  {...pages}
</Pages>
```
